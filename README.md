# Sistema de Assinaturas - Java

Projeto desenvolvido com o objetivo de praticar Programação Orientada a Objetos em Java.

## Conceitos aplicados
- Classes abstratas
- Polimorfismo
- Encapsulamento
- Regras de negócio centralizadas
- Enum

## Descrição
O sistema simula diferentes tipos de assinaturas:
- Básica
- Premium (com desconto)
- Empresarial (com desconto e taxa)

Cada tipo de assinatura possui regras próprias para o cálculo do valor final.

## Como rodar
- Clone o repositório na sua IDE ou editor de código preferido.
- Abra a pasta src.
- Rode o main.
